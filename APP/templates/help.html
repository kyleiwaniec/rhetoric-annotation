Instructions

Given a sentence to annotate (2), select a group (3) and property (4) on the left. The ChatGPT response (5) can speed up the process, but isn’t always correct, so should always be validated by the annotator.

Required: select a node in the parse tree (1) corresponding to the annotation value (6).

Select the appropriate checkbox(es) (6).

Click the Add Property (7) button.

Click the Next Property (8) button or select another property (4) from the list on the left.

Select the next annotation group (3).

Finally, mark the sentence as completed by clicking the Mark Completed (9) button. The sentence will no longer be presented for annotation, but will be available under the “completed” (12) tab on the Review page. If the Mark Completed button is not clicked, the current sentence will appear as “in-progress” (11) on the Review page and will remain in the pool of sentences to be annotated. You can always go back to any sentence from the Review page.

Click the Next Sentence (10) button to proceed.
